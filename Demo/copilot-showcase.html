🎤 Cedra Live Demo — Grant Application
Live on stage in Tromsø: one prompt → research → analysis → PDF → email, all in 5 minutes
Playwright Web Research PDF Generation Outlook Integration Live Demo
Presented at the Cedra opening in Tromsø, March 2026 for an SMB customer audience. A single prompt asked the AI to research a ski lift foundation (Sparebankstiftelsen SpareBank 1 Nord-Norge), analyze grant chances for BakOlsen skitrekk, write a formal application, and email the PDF — all live on stage.
$ copilot
> Jeg skal ha møte med Kristina Antonsen i Sparebankstiftelsen SpareBank 1 Nord-Norge. Jeg sitter i styret til BakOlsen skitrekk og vi trenger støtte til ny heis. Research Sparebankstiftelsen og BakOlsen, vurder sjansen for å få støtte, hvor mye vi bør be om, hva er risikoen, hvorfor vil søknaden bli godkjent og hvorfor ikke. Lag søknaden som en PDF og send den til meg når den er ferdig.
  • Step 1: Playwright opens Sparebankstiftelsen website → reads grant guidelines, amounts, deadlines
  • Step 2: Playwright opens BakOlsen.no → reads history, current infrastructure, user numbers
  • Step 3: Playwright opens Brønnøysundregistrene → fetches official organization data
  • Step 4: AI analyzes fit, estimates approval probability, recommends amount
  • Step 5: Generates formal PDF grant application
  • Step 6: Sends completed PDF via Outlook email

📂 Includes: reveal.js slide deck, teleprompter app, cached research data, backup PDF

Iterative UX with User Feedback

Visual evidence of AI iteration
📱 Piggtind Mobile — 5 Rounds of Feedback
User sends screenshot from phone → AI adjusts CSS → repeat until perfect
The Piggtind trip detail page went through 5 rounds of user feedback on real iPhone screenshots. Each round: user takes a photo, describes what's wrong, AI fixes the CSS, user tests again. Final result: map-first layout with compact overlay, iOS fullscreen fallback, thumb-friendly scroll area.
  • Round 1: "Map is too small" → changed to map-first layout filling viewport
  • Round 2: "Overlay hides the peak" → made overlay static, positioned above map
  • Round 3: "Need more scroll grab area" → increased grab area to 80px
  • Round 4: "Toolbar overlaps zoom controls" → offset MapLibre controls by 54px
  • Round 5: "Fullscreen doesn't work on iPhone" → CSS-based fullscreen fallback for iOS Safari
